# RFC-0002: WhatsApp Group Personality

## Status

Draft

## Summary

This feature allows each WhatsApp group connected to Hermes to have its own personality, tone, rules and contextual behavior.

## Motivation

Different groups have different purposes. Hermes should adapt its behavior depending on the group where it is operating.

## Initial Scope

- Define a personality per WhatsApp group
- Store tone and behavior instructions
- Apply the correct personality when responding in a group
- Allow personality changes by authorized users

## Out of Scope

- Advanced memory management
- Full admin dashboard
- Complex permission hierarchy
- Personality marketplace

## Open Questions

- Where will group configuration be stored?
- Who can edit a group personality?
- Should Hermes have a default personality?
- Should personalities support memory rules?

## Acceptance Criteria

To be defined.
