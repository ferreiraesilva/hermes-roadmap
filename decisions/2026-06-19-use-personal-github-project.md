# Use Personal GitHub Project for Hermes Roadmap

## Date

2026-06-19

## Status

Accepted

## Context

The Hermes ecosystem will include multiple repositories, including products, modules and experiments.

A GitHub Organization would be useful in the future, but the current setup will start under a personal GitHub account.

## Decision

Use a personal GitHub Project to centralize the Hermes ecosystem roadmap.

Create a `hermes-roadmap` repository to store product vision, RFCs, decisions, templates and cross-repository planning.

## Consequences

- The roadmap can start immediately without requiring a paid organization.
- Repositories can remain under the personal account.
- Collaborators must be added individually to each repository.
- The structure can be migrated to an organization later if the ecosystem grows.
